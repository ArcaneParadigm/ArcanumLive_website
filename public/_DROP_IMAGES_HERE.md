# HOW TO ADD FILES — just drop and go

No config. No code changes. Drop a file in the right folder → it appears on the site.

═══════════════════════════════════════════════════════════

## 🖼️ Realm images  →  public/realms/[realm-name]/

  public/realms/creatrix/            ← main gallery (drop anything here)
  public/realms/creatrix/gallery/    ← same, works as a subfolder too
  public/realms/creatrix/lore/       ← lore section (auto-split across chapters)
  public/realms/creatrix/characters/ ← character portraits

  Same structure for every realm:
    the-arcanum / mythmachine / soulblade / atlantis-nexus /
    gaia-heart-shard / aether / creatrix / raven /
    kali / coyote / dome-worlds / cosmic-surfer

  Supported image formats: jpg  jpeg  png  webp  avif  gif

═══════════════════════════════════════════════════════════

## 🎵 Realm audio  →  public/audio/realms/[realm-name]/

  Drop mp3 / wav / ogg / flac / m4a files here.
  They become the realm's playlist automatically.

  Filename tip: start with a number so they sort correctly:
    01-opening-theme.mp3
    02-the-descent.mp3
    03-sacred-geometry.mp3

  The filename (minus the extension) becomes the track title on the player.
  No metadata editing needed.

═══════════════════════════════════════════════════════════

## Other folders

  public/dome-shows/   ← dome show imagery
  public/music/        ← music page assets
  public/store/        ← store page assets

═══════════════════════════════════════════════════════════

That's it. Save the file, refresh the page.
