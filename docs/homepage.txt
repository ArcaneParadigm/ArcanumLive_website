Arcanum.Live Homepage Copy + Design Doc
Homepage Core Concept
The homepage should not feel like a normal website. It should feel like the visitor is entering the metaverse of The Arcanum.
The first page is a portal chamber: image-heavy, cinematic, interactive, animated, musical, and alive. The world itself should act like a map. The main pathways should be highlighted clearly so visitors know where to go.
Core feeling:
You are entering The Arcanum.
The homepage is a portal chamber.
The image is the map.
The buttons are gateways.
The cursor awakens the world.
The scroll pulls you deeper.
Main homepage flow:
1. Portal Entrance
2. Dome Shows
3. 360 Movie Rentals
4. The Worlds
5. Screensaver / Living Visualizer
6. Archive / Vault
7. Music / Sonic Realms

2. Simple Hero Layer Structure
Build the homepage hero like stacked layers, so the main image/video can be swapped later without rebuilding the whole page.
LAYER 1  = main background image OR looping video
LAYER 2  = optional dark gradient / readability overlay
LAYER 3  = smoke / fog overlay
LAYER 4  = particles / drifting embers / stars
LAYER 5  = portal glow / ring animation
LAYER 6  = title + subtitle
LAYER 7  = gateway buttons
LAYER 8  = invisible hotspot zones
LAYER 9  = sound toggle / top nav / scroll cue
Recommended V1:
Main hero = one strong background image
Optional center portal loop = short looping video or animated SVG/CSS ring
Smoke = separate overlay
Particles = separate overlay
Buttons = separate UI layer
Hotspots = invisible positioned zones
Best hybrid setup:
Background = still image
Center portal = looping video or animated ring layer
Smoke = transparent looping overlay
Particles = canvas or CSS overlay
Character motion = optional later
This gives a living-world feeling while keeping image replacement simple.

5. Cursor Parallax Behavior
Use mouse/pointer movement to create subtle cinematic depth.
Rules:
background moves least
foreground pillars move moderately
guardian character moves slightly more
floating glyphs move most
buttons move very little for readability
portal ring rotates independently
Recommended movement range:
background: 4-8px
midground: 8-16px
foreground: 12-24px
particles/glyphs: 20-40px
Desktop behavior:
cursor position controls parallax offset
hovering hotspots intensifies the related area
Mobile behavior:
use slow idle parallax
no required hover interactions
buttons remain visible and tappable
Accessibility:
respect prefers-reduced-motion
allow motion intensity setting later
keep parallax subtle to avoid nausea

8. Main Video / Looping Image Behavior
The hero can use a looping movie as the main image, but the best V1 approach is hybrid.
Option A: Static hero image + animated overlays
Best for:
sharp artwork
easier swapping
better performance
clean hotspot mapping
Option B: Looping video as main hero
Best if:
the loop is high quality
file size is controlled
motion is slow and atmospheric
no important text is baked into it
poster fallback exists
Good loop content:
portal rotating
smoke drifting
character hair moving
cloth fluttering
magical energy pulses
slow camera float
Recommended V1:
static high-resolution key art
animated overlay layers
looping portal/ring animation
particle/smoke overlays
optional small video loop inside the portal

9. Scroll Behavior
Use standard scroll as the primary journey mechanic.
Do not make left mouse drag the required way to enter the image.
Recommended pattern:
sticky hero with scroll-driven portal zoom
Scroll stage A: Arrival
full scene visible
title visible
buttons visible
portal slowly spins
ambient particles drift
Scroll stage B: Awakening
As user scrolls slightly:
portal enlarges 5-10%
foreground layers separate
floating glyphs accelerate slightly
button row subtly lowers or fades
edge vignette increases
Scroll stage C: Threshold
At mid-scroll:
camera pushes toward central portal
portal glow brightens
rings rotate faster
world darkens around edges
short text appears: Choose Your Path or Step Deeper Into The Arcanum
Scroll stage D: Release
After pinned hero sequence:
hero unpins
page scrolls into next immersive panel
next section begins with Dome Shows or 360 Movies
Optional:
clicking central portal can jump to /screensaver or open a Choose Your Path overlay
